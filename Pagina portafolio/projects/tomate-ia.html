IA aplicada

Sistema de detección de enfermedades en plantas de tomate mediante inteligencia artificial

Desarrollo de un sistema de clasificación de imágenes enfocado en identificar enfermedades en hojas de tomate y asociar una recomendación útil a partir del resultado obtenido por el modelo.

Objetivo

Desarrollar un sistema capaz de analizar imágenes de hojas de tomate para identificar posibles enfermedades y ofrecer una respuesta útil basada en la categoría detectada.

La intención del proyecto fue aplicar inteligencia artificial a un problema concreto, combinando clasificación automática, análisis visual e interpretación del resultado.

Mi aporte

En este proyecto trabajé en la preparación del flujo general del sistema, el entrenamiento del modelo y la parte de inferencia sobre nuevas imágenes.

  • Preparación y organización del dataset
  • Preprocesamiento y aumento de imágenes
  • Construcción y entrenamiento del modelo CNN
  • Evaluación del desempeño del sistema
  • Predicción y asociación de recomendaciones por clase

Desarrollo del proyecto

El sistema fue construido a partir de un conjunto de imágenes organizadas por categorías, correspondientes a diferentes estados de salud de hojas de tomate. Para mejorar el aprendizaje del modelo, se trabajó con escalado, transformaciones y aumento de datos.

A nivel de arquitectura se utilizó una red convolucional para aprender patrones visuales relevantes y clasificar cada imagen dentro de un conjunto de categorías definidas.

Aspectos destacados

  • Aplicación real de IA en clasificación de imágenes
  • Uso de una CNN para reconocimiento de patrones visuales
  • Trabajo con imágenes, entrenamiento y validación
  • Salida útil para el usuario a partir de la predicción
  • Proyecto que fortalece mi perfil técnico y analítico

Resultado

El resultado fue un sistema funcional capaz de clasificar imágenes de hojas de tomate y entregar una interpretación asociada al tipo de enfermedad detectada o al estado saludable de la planta.

Este proyecto demuestra mi capacidad para trabajar con inteligencia artificial aplicada, procesamiento de imágenes y construcción de soluciones orientadas a problemas concretos.

Aprendizajes

Este trabajo fortaleció mis habilidades en análisis de datos visuales, entrenamiento de modelos, lógica de sistemas y comprensión del flujo entre datos de entrada, predicción y salida útil para el usuario.

También me permitió desarrollar una visión más práctica sobre cómo la inteligencia artificial puede integrarse a soluciones con valor real.
